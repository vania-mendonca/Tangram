Le InsTangram 3D

Three-Dimensional Visualization and Animation Project

Download this project as a .zip file Download this project as a tar.gz file

Concept

Le InsTangram 3D is an interactive application where the user can build 3D figures with the seven Tangram pieces. These pieces can be translated or rotated, and their reflection appears on a mirror placed on top of a wood table.

It is possible to view the scene from different points of view and apply image filters (individually or a combination of several filters). It is also possible to save the current figure configuration in order to access the next time one opens the application, as well as to save a screenshot of the scene at a given moment.
